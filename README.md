# DDS Viewer - VSCode扩展

![DDS Viewer Logo]([icon.png](https://raw.gitcode.com/StarsAll_InFrost/dds-viewer/blobs/53ee76b835e5aee9f6e1db04cd266850341b8231/icon.png))

一个专业的VSCode扩展，用于预览和导出DDS（DirectDraw Surface）图像文件。支持多种DDS格式，提供直观的UI界面和丰富的图像操作功能。

## ✨ 主要特性

### 🖼️ 图像预览
- **实时预览**：直接在VSCode中查看DDS纹理
- **多格式支持**：支持DXT1、DXT3、DXT5、BC4、BC5等压缩格式
- **高质量渲染**：保持原始图像质量进行预览

### 🔍 交互操作
- **缩放控制**：支持放大、缩小、重置和适应视图
- **拖拽查看**：鼠标拖拽查看大尺寸纹理
- **实时信息**：显示详细的DDS文件头信息

### 📤 导出功能
- **PNG导出**：将DDS纹理导出为PNG格式
- **批量处理**：支持多个DDS文件的预览和导出
- **保持质量**：导出时保持原始分辨率和质量

### 📊 详细分析
- **格式检测**：自动识别DDS压缩格式
- **头信息解析**：显示完整的DDS文件头数据
- **技术参数**：展示尺寸、Mipmap、FourCC等关键信息

## 🚀 快速开始

### 安装扩展
1. 在VSCode扩展商店中搜索"DDS Viewer"
2. 点击安装按钮
3. 重启VSCode

### 使用方法
1. **方法一**：在VSCode中直接打开.dds文件
2. **方法二**：使用命令面板（Ctrl+Shift+P）输入"DDS Viewer: Preview DDS File"
3. **方法三**：在资源管理器中右键点击.dds文件，选择"Open With" → "DDS Preview"

| 格式 | 描述 | 状态 |

| DXT1 | BC1压缩格式 | ✅ 完全支持 |
| DXT3 | BC2压缩格式 | ✅ 完全支持 |
| DXT5 | BC3压缩格式 | ✅ 完全支持 |
| BC4 | ATI1压缩格式 | ✅ 完全支持 |
| BC5 | ATI2压缩格式 | ✅ 完全支持 |

| BGR | 未压缩24位 | ✅ 完全支持 |
| DX10 | 扩展头格式 | 🔄 部分支持 |
